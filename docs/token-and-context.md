# Token use (free edition)

The free core adds roughly **~1–2k tokens/turn** of always-on rules — modest on modern context windows.

## Keep costs down

1. **Start with rules 00, 02, 03 only** — add 01, 04, 07 when needed (see README install section).
2. **Evidence:** paste real command output; for long logs use summary + last ~25 lines (see docs/agent-operating-contract.md).
3. **No Team OS** in free edition — multi-role chains are the main token multiplier.

The full Pro + Team OS guide is included in the Team edition zip (docs/token-and-context.md).
